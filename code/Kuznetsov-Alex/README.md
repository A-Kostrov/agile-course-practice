# Расчет площади поверхности трехмерных фигур

Выполнил:

 - Кузнецов Александр
 - ННГУ, ИИТММ, каф. МОСТ, группа M0816-1

## Задание

Реализовать расчёт площади поверхности наиболее известных трёхмерных фигур:

 - Сфера
 - Цилиндр
 - Конус
 - Тетраэдр
 - Октаэдр
 - Икосаэдр
 - Гексаэдр(куб)
