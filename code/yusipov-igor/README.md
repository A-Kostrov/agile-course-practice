## Задание

Реализовать класс конвертора температур с поддержкой конвертации температур из шкалы Цельсия в шкалы:

 - Фаренгейт
 - Кельвин
 - Ньютон

 - ["Кельвин"][kelvin] на сайте Wikipedia
